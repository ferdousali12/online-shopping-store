# ESSENCE Perfume Shop

A responsive perfume and cosmetics e-commerce website built with HTML, CSS, and JavaScript.

## Live Demo

[View Live Website](https://onlineshoppingshore.netlify.app/)

## Project Description

ESSENCE Perfume Shop is an online shopping website for perfumes, cosmetic products, male perfumes, and makeup items. The project is designed like a small e-commerce store where users can explore product categories, view product cards, add products to the cart, and review their selected items on a cart page.

This project helped me practice frontend development skills, including creating responsive layouts, designing product sections, adding JavaScript interactivity, and managing cart data using browser local storage.

## Features

- Responsive homepage and dashboard
- Video hero section
- Product category cards
- Perfume container section
- Cosmetic container section
- Male perfume container section
- Makeup container section
- Product cards with images, names, prices, and old prices
- Add to cart functionality
- Cart item quantity increase and decrease
- Remove item from cart
- Clear cart option
- Order summary with subtotal, shipping, tax, and total
- Search functionality
- Smooth scrolling between sections
- Mobile-friendly navigation menu

## Pages Included

- `index.html` - Landing/login page
- `dashboard.html` - Main shopping page with product categories
- `cart.html` - Shopping cart page
- `style.css` - Website styling and responsive design
- `script.js` - JavaScript functionality

## What I Learned

While building this project, I learned how to structure a multi-page website and connect different pages together. I practiced creating product sections, designing reusable product cards, and making the layout responsive for different screen sizes.

I also learned how to use JavaScript to make the website interactive. I implemented add-to-cart functionality, updated cart quantities, removed products, cleared the cart, and stored cart data using localStorage so the cart data stays saved in the browser.

This project also helped me understand how an e-commerce website works on the frontend, including product browsing, category navigation, cart management, and order summary calculation.
